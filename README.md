# Test The Movie DB

A simple app to play with the API of The Movie DB website!

## Install

```
cp .env .env.local
```

## What do you find in this app?

- [x] 404
- [x] Absolute Imports
- [x] Axios
- [x] MUI
- [x] Prettier
- [x] React Router
- [x] Route-based code splitting
- [x] SCSS
- [x] Tests
- [x] TypeScript
