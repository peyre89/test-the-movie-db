# Test The Movie DB

A simple app to play with the API of The Movie DB website!

## Install

```
cp .env .env.local
```

## What do you find in this app?

- [x] Absolute Imports
- [x] Axios
- [x] Axios Hooks
- [x] Cache (Axios Hooks uses LRU)
- [x] MUI (UI library)
- [x] Page 404
- [x] Prettier
- [x] React Helmet
- [x] React Router
- [x] React Redux
- [x] Route-based code splitting
- [x] SCSS
- [x] Tests
- [x] TypeScript
